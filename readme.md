Just trying out typescript

### install typescript

```
sudo npm i -g typescript
```

```
$ tsc -v
Version 5.2.2
```

### convert typescript to javascript

```
tsc index.tx
```

### generate tsconfig file

```
tsc --init
```

### automatically transpile

```
tsc --watch
```

### typescript with react

```
npm create vite@latest
cd app
npm install
# fix module errors if present
npm run dev
```
